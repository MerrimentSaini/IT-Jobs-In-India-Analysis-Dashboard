# 📊 IT Jobs Analysis Dashboard (Power BI)

## 📌 Project Overview

The **IT Jobs Analysis Dashboard** is an interactive Power BI project that analyzes the Indian technology job market using real-world job posting data.

This dashboard transforms raw data into meaningful business insights, helping users understand hiring trends, salary distribution, company ratings, work modes, experience requirements, skills in demand, and job locations.

---

# 🎯 Project Objectives

- Analyze hiring trends across India.
- Identify top hiring companies.
- Compare salary ranges across cities and job roles.
- Explore experience requirements for different jobs.
- Analyze company ratings and company sizes.
- Identify the most in-demand skills.
- Build an interactive dashboard for better decision-making.
